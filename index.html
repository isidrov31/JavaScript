Scope

Definicion:

  • Scope Global:Las variables declaradas fuera de cualquier función tienen un alcance global. Esto significa que son accesibles desde cualquier parte del código, ya sea dentro de funciones, bucles o bloques de código.
  • Scope Local:Las variables declaradas dentro de una función tienen un alcance local. Esto significa que solo pueden ser accedidas desde dentro de la función donde se declaran. Estas variables no están disponibles fuera de la función y se eliminan de la memoria una vez que la función finaliza su ejecución.
  • JavaScript también tiene scopes de bloque introducidos por la declaración let y const en ES6. Las variables declaradas con let y const tienen un alcance de bloque, lo que significa que solo son accesibles dentro del bloque de código donde se declaran.

Switch

Definicion:

La declaración switch se utiliza para realizar una selección múltiple basada en el valor de una expresión. Permite ejecutar diferentes bloques de código dependiendo del valor que coincida con los case especificados.
  • Expression: es la expresión que se evalúa y se compara con los case para determinar qué bloque de código se ejecutará. Puede ser cualquier expresión que produzca un valor, como una variable, una constante o una llamada a una función.
  • Case: es una etiqueta que representa un valor específico con el cual se compara la expresión. Si la expresión coincide con el valor del case, se ejecutará el bloque de código asociado a ese case.
  • Break: Después de cada bloque de código en un case, se utiliza la palabra clave break para salir del switch. Esto evita que se ejecuten los bloques de código de los case siguientes. Si se omite el break, la ejecución continuará hacia el siguiente case, incluso si la expresión no coincide con su valor.
  • Default:El bloque de código default se ejecuta si ninguno de los case coincide con el valor de la expresión. Es opcional y se utiliza para proporcionar un comportamiento predeterminado cuando no hay coincidencias.

For

Definicion:

Crea un bucle que consiste en tres expresiones opcionales, encerradas en paréntesis y separadas por puntos y comas, seguidas de una sentencia ejecutada en un bucle.

for ([expresion-inicial]; [condicion]; [expresion-final])sentencia

  • Expresion-inicial:Una expresión (incluyendo las expresiones de asignación) o la declaración de variable. Típicamente se utiliza para usarse como variable contador.
  • Condicion: Una expresión para ser evaluada antes de cada iteración del bucle. Si esta expresión se evalúa como verdadera, se ejecuta sentencia. Esta comprobación condicional es opcional. Si se omite, la condición siempre se evalúa como verdadera. Si la expresión se evalúa como falsa, la ejecución salta a la primera expresión que sigue al constructor de for.
  • Expresion-final: Una expresión para ser evaluada al final de cada iteración del bucle. Esto ocurre antes de la siguiente evaluación de la condicion. Generalmente se usa para actualizar o incrementar la variable contador.
  • Sentencia:se ejecuta mientras la condición se evalúa como verdadera.

Objetos

Definicion:

En JavaScript, un objeto es un entidad independiente con propiedades y tipos.Un objeto de JavaScript tiene propiedades asociadas a él. Una propiedad de un objeto se puede explicar como una variable adjunta al objeto. Las propiedades de un objeto básicamente son lo mismo que las variables comunes de JavaScript, excepto por el nexo con el objeto. Las propiedades de un objeto definen las características del objeto. Accedes a las propiedades de un objeto con una simple notación de puntos: objectName.propertyName

Creación de objetos mediante objetos literales: Puedes crear objetos directamente utilizando objetos literales, que son simplemente una lista de pares clave-valor encerrados entre llaves {}. Cada clave es una propiedad del objeto y cada valor es el valor asignado a esa propiedad. Aquí tienes un ejemplo:
const miObjeto = {
propiedad1: valor1,
propiedad2: valor2,
};

Creación de objetos utilizando el constructor Object: Creación de objetos utilizando el constructor Object: JavaScript proporciona el constructor Object que se puede utilizar para crear objetos vacíos o agregar propiedades a un objeto existente. Aquí tienes un ejemplo:
const miObjeto = new Object();
miObjeto.propiedad1 = valor1;
miObjeto.propiedad2 = valor2;

Creación de objetos utilizando una función constructora: Puedes definir una función constructora y luego crear nuevos objetos utilizando esa función con la palabra clave new. Dentro de la función constructora, puedes definir propiedades y métodos para los objetos creados. Aquí tienes un ejemplo:
function MiObjeto(propiedad1, propiedad2) {
this.propiedad1 = propiedad1;
this.propiedad2 = propiedad2;
// ...
}
const miNuevoObjeto = new MiObjeto(valor1, valor2);

Clases

Definicion:

Una clase en JavaScript actúa como una plantilla o un plano para crear objetos. Define propiedades y métodos que serán compartidos por todas las instancias de objetos creadas a partir de esa clase. Aquí tienes un ejemplo de cómo se crea una clase en JavaScript:

class MiClase {
constructor(propiedad1, propiedad2) {
this.propiedad1 = propiedad1;
this.propiedad2 = propiedad2;
}

Para crear un objeto a partir de una clase, se utiliza la palabra clave new seguida del nombre de la clase y los argumentos necesarios para el constructor, si los hay:

const miObjeto = new MiClase(valor1, valor2);

Closures

Definicion:

es una combinación de una función y el entorno léxico en el cual se declaró esa función. Un closure permite que una función acceda y conserve referencias a variables y parámetros externos, incluso después de que la función haya finalizado su ejecución.

function exterior() {
var mensaje = 'Hola';

function interior() {
console.log(mensaje);
}

return interior;
}

var miFuncion = exterior();
miFuncion(); // Imprime 'Hola'

En este ejemplo, la función interior es un closure. Aunque se ejecuta fuera de la función exterior, aún tiene acceso a la variable mensaje que se declaró en el ámbito de exterior. Al llamar a exterior(), se devuelve la función interior y se asigna a miFuncion. Luego, al llamar a miFuncion(), se imprime 'Hola'. El closure permite que interior conserve el entorno léxico en el que se declaró, incluyendo todas las variables y parámetros que estaban disponibles en ese momento. Esto es útil cuando se necesitan funciones que recuerden ciertos valores o estados incluso después de que la función externa haya terminado su ejecución.
