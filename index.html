Tipos de Variables en JavaScript

Tipo de Variable Descripción Ejemplo
Number Representa números, ya sea enteros o decimales let edad = 25;
String Representa cadenas de texto let nombre = "Juan";
Boolean Representa un valor lógico: verdadero (true) o falso (false) let esMayorDeEdad = true;
Array Representa una colección ordenada de valores let numeros = [1, 2, 3, 4, 5];
Object Representa una entidad con propiedades y métodos let persona = { nombre: "Ana", edad: 30 };
Null Representa la ausencia intencional de cualquier objeto o valor let resultado = null;
Undefined Representa una variable que no ha sido asignada let dato;

El Prompt

El prompt es un método en JavaScript que muestra un cuadro de diálogo modal en el navegador, donde el usuario puede ingresar texto.

Alert

El método Window.alert() muestra un diálogo de alerta con un mensaje opcional, y aguardará hasta que el usuario cierre la ventana de diálogo.

Tipos de Operadores en JavaScript

Tipo de Operador Significado
Operadores Aritméticos Se utilizan para realizar operaciones matemáticas en JavaScript.
+ Suma dos valores.
- Resta el segundo valor del primero.
* Multiplica dos valores.
/ Divide el primer valor por el segundo.
% Devuelve el resto de la división entera de un valor por otro.
Operadores de Asignación Se utilizan para asignar valores a variables en JavaScript.
= Asigna el valor de la expresión a la variable.
+= Incrementa el valor de la variable en la cantidad especificada.
-= Decrementa el valor de la variable en la cantidad especificada.
*= Multiplica el valor de la variable por la cantidad especificada y lo asigna.
/= Divide el valor de la variable por la cantidad especificada y lo asigna.
Operadores de Comparación Se utilizan para comparar valores en JavaScript.
== Comprueba si dos valores son iguales.
=== Comprueba si dos valores son iguales en valor y tipo.
!= Comprueba si dos valores son diferentes.
!== Comprueba si dos valores son diferentes en valor o tipo.
> Comprueba si el primer valor es mayor que el segundo.
< Comprueba si el primer valor es menor que el segundo.
>= Comprueba si el primer valor es mayor o igual que el segundo.
<= Comprueba si el primer valor es menor o igual que el segundo.
Operadores Lógicos Se utilizan para realizar operaciones lógicas en JavaScript.
&& Evalúa si ambas condiciones son verdaderas.
|| Evalúa si al menos una de las condiciones es verdadera.
! Devuelve el valor opuesto de la condición (negación lógica).

Scope

  • Scope Global:Las variables declaradas fuera de cualquier función tienen un alcance global. Esto significa que son accesibles desde cualquier parte del código, ya sea dentro de funciones, bucles o bloques de código.
  • Scope Local:Las variables declaradas dentro de una función tienen un alcance local. Esto significa que solo pueden ser accedidas desde dentro de la función donde se declaran. Estas variables no están disponibles fuera de la función y se eliminan de la memoria una vez que la función finaliza su ejecución.
  • JavaScript también tiene scopes de bloque introducidos por la declaración let y const en ES6. Las variables declaradas con let y const tienen un alcance de bloque, lo que significa que solo son accesibles dentro del bloque de código donde se declaran.

If Else

Ejecuta una sentencia si una condición específicada es evaluada como verdadera. Si la condición es evaluada como falsa, otra sentencia puede ser ejecutada

Tipos de Datos JavaScript

Tipo de Dato Descripción
Tipos de Datos Primitivos Representan valores simples en JavaScript.
Number Representa números, tanto enteros como decimales.
String Representa una secuencia de caracteres.
Boolean Representa un valor lógico: verdadero o falso.
Null Representa la ausencia intencional de cualquier objeto o valor.
Undefined Representa una variable que no tiene un valor asignado.
Objetos Representan una colección de propiedades relacionadas.
Array Representa una lista ordenada de valores.
Object Representa un objeto con propiedades y métodos.
Function Representa un bloque de código reutilizable.
Tipos de Datos Especiales Representan valores especiales en JavaScript.
Symbol Representa un valor único e inmutable que se utiliza como identificador de propiedad.
